# Maintenance Intelligence v2

## Overview
This project is a simple risk detection system based on trend analysis of equipment data (e.g., temperature).

## Features
- Trend calculation
- Risk classification (High, Medium, Normal)
- Basic test cases

## Purpose
This project simulates how data-driven systems can detect early warning signs in mechanical or infrastructure systems.

## Future Improvements
- Add pressure and vibration analysis
- Implement rolling averages
- Add anomaly detection
- Integrate real maintenance logs (CSV)
